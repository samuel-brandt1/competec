# Competec 2025
Promovido pelo Programa de Educação Tutorial em Sistemas de Informação (PET-SI) e sob a coordenação geral do Prof. Marcelo Medeiros Eler, o projeto tem como foco o ensino de lógica e técnicas de programação para alunos do ensino médio técnico das ETECs da zona leste de São Paulo. O nome COMPETEC une os elementos-chave da iniciativa: COMputação + PET + ETEC.
